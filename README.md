# Nx monorepo templates

## Installation

```shell
$ git clone https://github.com/jungpaeng/nx-monorepo-templates.git
$ cd nx-monorepo-templates
$ yarn
```

## Add new Package

### Vanilla JavaScript

```shell
$ yarn new-package
? 패키지 이름을 입력해주세요. › utils
? 패키지 설명을 입력해주세요. › 유틸입니다
```

### React

```shell
$ yarn new-react-package
? 패키지 이름을 입력해주세요. › utils
? 패키지 설명을 입력해주세요. › 유틸입니다
```
